三、文件及目录

_config.yml

网站的配置信息，全局配置文件

package.json

应用程序信息，执行npm install安装hexo相关程序依赖的。安装后可以删除。

scaffolds

模板文件夹。

scripts

脚本文件夹。

source

资源文件夹。

themes

主题文件夹。Hexo会根据主题来生成静态页面。
